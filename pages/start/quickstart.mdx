<Info>
  Este guia rápido te ajudará a fazer sua primeira requisição à API da
  4SeletPay em poucos minutos.
</Info>

## Pré-requisitos

Antes de começar, você precisa de:

- Uma conta na 4SeletPay
- Uma chave de API (criada no dashboard)
- Conhecimento básico de HTTP e JSON

<Tip>
  Se você ainda não tem uma conta, acesse o <a href="https://app.4seletpay.com.br" target="_blank">dashboard</a> e crie sua conta gratuitamente.
</Tip>

## Passo 1: Obter sua Chave de API

Primeiro, você precisa criar uma chave de API no dashboard da 4SeletPay:

<Steps>
  <Step title="Acesse o Dashboard">
    Faça login no <a href="https://app.4seletpay.com.br" target="_blank">dashboard da 4SeletPay</a>
  </Step>
  <Step title="Crie uma chave de API">
    Navegue até a seção de integração e clique em "Criar Chave"
  </Step>
  <Step title="Copie sua chave">
    Copie a chave gerada e armazene-a em um local seguro
  </Step>
</Steps>

<Warning>
  **Importante**: Sua chave de API é uma credencial sensível. Nunca a
  compartilhe publicamente ou a commite em repositórios de código.
</Warning>

<Note>
  O modo da cobrança (dev ou produção) é determinado automaticamente pela **chave de API** utilizada na requisição:

  - **Chave de API de Dev**: Transações simuladas, ideal para testes
  - **Chave de API de Produção**: Transações reais, requer conta verificada

  Observe que na resposta, o campo `devMode` indica se a cobrança foi criada em modo de desenvolvimento ou produção.
</Note>

Para mais detalhes sobre autenticação, consulte a [documentação de autenticação](/pages/authentication).

## Passo 2: Fazer sua Primeira Requisição

Agora vamos criar sua primeira cobrança de Checkout Transparente via API.

### Exemplo com cURL

```bash
curl --request POST \
  --url https://app.4seletpay.com.br/api/v1/transparents/create \
  --header 'Authorization: Bearer {{API_KEY}}' \
  --header 'Content-Type: application/json' \
  --data '{
    "amount": 1000,
    "method": "PIX_QRCODE"
  }'
```
