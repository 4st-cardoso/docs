## Visão Geral

Os endpoints de **Gestão** oferecem acesso a relatórios consolidados da plataforma. O acesso é restrito a usuários com perfil de **Super Admin**.

---

## Relatório de Vendas

O endpoint `GET /v1/management/sales-report` retorna dados agregados de vendas de todas as contas, com suporte a filtros de data e paginação.

### Campos do relatório

| Campo              | Tipo    | Descrição                               |
|--------------------|---------|-----------------------------------------|
| `account_code`     | string  | Código da conta                         |
| `account_name`     | string  | Nome da conta                           |
| `total_revenue`    | float   | Receita total no período                |
| `total_orders`     | integer | Total de pedidos                        |
| `approved_orders`  | integer | Pedidos aprovados                       |
| `declined_orders`  | integer | Pedidos recusados                       |
| `period.start_date`| date    | Data inicial do filtro                  |
| `period.end_date`  | date    | Data final do filtro                    |

### Filtros disponíveis

| Parâmetro    | Tipo    | Descrição                                        |
|--------------|---------|--------------------------------------------------|
| `start_date` | date    | Data inicial (formato Y-m-d)                     |
| `end_date`   | date    | Data final (deve ser ≥ start_date, formato Y-m-d)|
| `page`       | integer | Número da página (mín. 1)                        |
| `per_page`   | integer | Itens por página (mín. 1, máx. 100)              |

<Warning>
  Acesso exclusivo para Super Admins. Tentativas de acesso sem a permissão correta resultam em erro 403.
</Warning>
