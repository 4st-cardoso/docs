Links de pagamento permitem criar cobranças que podem ser pagas **múltiplas vezes**. Ideal para vendas onde vários clientes pagam pelo mesmo link.

## Criar link de pagamento

Use `/checkouts/create` com `frequency: "MULTIPLE_PAYMENTS"`.

<Info>
  A diferença para um checkout comum é o parâmetro `frequency` definido como `MULTIPLE_PAYMENTS`.
</Info>

**Exemplo:**
```json
POST /checkouts/create
{
  "items": [
    {
      "id": "prod_pro",
      "quantity": 1
    }
  ],
  "frequency": "MULTIPLE_PAYMENTS",
  "methods": ["PIX", "CARD"]
}
```

**Resposta:**
```json
{
  "data": {
    "id": "bill_abc123xyz",
    "url": "https://app.4seletpay.com.br/pay/bill_abc123xyz",
    "amount": 10000,
    "status": "PENDING",
    "frequency": "MULTIPLE_PAYMENTS",
    "items": [{ "id": "prod_pro", "quantity": 1 }],
    "coupons": [],
    "devMode": false,
    "createdAt": "2024-11-04T18:38:28.573Z",
    "updatedAt": "2024-11-04T18:38:28.573Z"
  },
  "success": true,
  "error": null
}
```

<Tip>
  Compartilhe a `url` com seus clientes. O link pode ser pago múltiplas vezes.
</Tip>
